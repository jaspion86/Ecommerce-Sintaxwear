# SyntaxWear - Tênis Sneakers Online

SyntaxWear é um projeto de e-commerce de tênis sneakers desenvolvido para demonstrar habilidades em HTML5 e CSS3. A página principal é uma landing page que apresenta os produtos e categorias da loja.

## Tecnologias Utilizadas

- HTML5
- CSS3

## Estrutura de Pastas

```
├───index.html
├───README.md
├───css
│   ├───base.css
│   ├───layout.css
│   ├───reset.css
│   ├───variables.css
│   └───components
│       ├───footer.css
│       ├───header.css
│       ├───hero.css
│       ├───product-category.css
│       └───product-grid.css
└───images
    ├───banners
    ├───favicons
    ├───icons
    ├───logo
    └───products
```

- **index.html**: A página principal da aplicação.
- **css**: Contém os arquivos de estilo da aplicação.
  - **components**: Contém os arquivos de estilo para os componentes específicos da página.
- **images**: Contém todas as imagens utilizadas na aplicação, como banners, ícones, logos e produtos.

## Como executar o projeto

Para executar o projeto, basta abrir o arquivo `index.html` em seu navegador de preferência.
